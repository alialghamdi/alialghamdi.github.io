# Lab 1: Setting up the environment

Objective:

To start the Big Data environment for the next labs.

Successful outcome:

  1. Imported the VM.
  2. Started the VM and services.

# Installing VirtualBox on Windows

  1. Find the installer provided by the instructor.

  2. Double click on the installer exe file.

  3. Progress by clicking on Next here:

  4. Click on next here as well:

  5. Click on next again:

  6. Click on yes when you see this warning:

  7. Click on Install to complete the installation:

  8. When you are asked whether to install the device, choose install:

  9. Click on Finish:

Voila! Now, VirtualBox should start and we will import the VM in this next step.

# Importing the Cloudera QuickStart VM

  1. Locate the Cloudera QuickStart VM shared by the instructor.

  2. Open VirtualBox if it's not already opened.

  3. Click on File -> Import:

  4. Choose the OVF file in the location shared by the instructor:

  5. When you get to appliance settings (Make processors 2 and RAM 10240) and then click on Import:

  6. It will take multiple minutes to import, you can check twitter now.

  7. You should see the VM on the left side, the one with the red hat, right click and then from Start choose Normal Start:

Now we have imported the VM we will be using all day for our exercises, after this we will start it and start the Hadoop ecosystem services and applications.

# Starting up the Cloudera QuickStart services

  1. After the VM has started, on the desktop, double click on Launch Cloudera Manager Express

  2. Wait for five-ten minutes and check if the service has started on the command line and the installation has been successful.

  3. Then access the Cloudera Manager UI through http://quickstart.cloudera:7180

	user: cloudera
