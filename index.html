# Intro to Big Data

Hello humans, this is built to help you in developing simple yet powerful knowledge in Big Data world and capabilities. We will get to know the infrastructure part i.e. Hadoop HDFS and YARN. And then, we will utilize the rest of Hadoop ecosystem to develop some interesting use-cases.

Using Cloudera CDH QuickStartVM we will be using one of the main Hadoop distributions in the market, utilizing this sandbox will help us to navigate easily between topics, since it's a full ecosystem provided in one sandbox.

There are many other use-cases that you can develop. What we will do here is just a very small introduction to the power of Big Data.
